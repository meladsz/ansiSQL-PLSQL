Descripción del proyecto

Este repositorio contiene los ejercicios y prácticas realizados durante el curso **Desarrollador Oracle SQL‑PL/SQL** impartido por **CAS Training**.  
Su objetivo es consolidar conocimientos en bases de datos Oracle mediante la práctica de **SQL** y **PL/SQL**, creando un recurso útil para consulta futura y desarrollo profesional.

---

Contenido del repositorio

- **ANSI‑SQL/**  
  Consultas básicas y avanzadas, joins, subconsultas, agrupaciones, funciones y ejercicios progresivos.

- **PLSQL/**  
  Procedimientos almacenados, funciones, cursores, excepciones, bucles y ejemplos aplicados a tablas EMP, DEPT, DOCTOR y HOSPITAL.

---

Objetivos del proyecto

- Dominar SQL y PL/SQL mediante ejercicios reales.  
- Comprender el modelo relacional y la manipulación de datos.  
- Crear procedimientos y funciones para automatizar tareas.  
- Desarrollar un repositorio organizado como herramienta profesional.
